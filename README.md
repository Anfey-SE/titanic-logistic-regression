# Titanic - Logistic Regression

Kaggle competition submission for "Titanic - Machine Learning from Disaster."

## Approach
- Cleaned missing values in Age, Fare, and Embarked
- Encoded categorical features (Sex, Embarked)
- Engineered a FamilySize feature from SibSp + Parch
- Trained a Logistic Regression model (scikit-learn)

## Result
- Validation accuracy: ~0.79
- Kaggle leaderboard score: 0.76315
